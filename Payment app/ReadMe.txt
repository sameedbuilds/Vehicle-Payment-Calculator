# 🚚 Vehicle Payment Calculator

A simple and user-friendly Streamlit web application designed to calculate vehicle payment transactions, especially for goods transportation businesses.


## Features

* Convert vehicle weight from **Kilograms (Kg)** to **Mund** automatically.
* Calculate the **buying cost** based on the broker rate.
* Add **fare charges** to determine the total amount spent.
* Calculate the **selling amount** based on the company rate.
* Automatically compute the **profit or loss** for each vehicle.
* Support for **multiple vehicle entries** in a single session.
* Display a complete summary of:

  * Total Amount Spent
  * Total Revenue
  * Total Profit/Loss
* Remove the last entry or clear all records with a single click.

## Technologies Used

* Python
* Streamlit

## Installation

1. Clone the repository:

```bash
git clone https://github.com/sameedbuilds/Vehicle_payment_calculator.git
```

2. Navigate to the project directory:

```bash
cd Vehicle_payment_calculator
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run Vehicle_payment_calculator.py
```
